Welcome to drf_template's documentation!
========================================

Template for django with django rest framework with some superpowers
make simple and fast to start a new app having a great configurations.

Some features are:

* Easy start project
* Fast develop a services
* 12 factor app
* Prepared for making docs with docstrings with Numpy/google format
